# FounderFocus CLI Toolkit

## Project Overview

This project is a command-line toolkit to help solo founders make data-driven startup decisions using real-world investment data. The toolkit performs startup analysis, calculates buzzword scores, categorizes funding stages, and identifies acquisition-ready startups.

## Files in This Repository

1. **founder_focus.py** - The Python module containing the `FounderFocus` class and related functions.
2. **founder_summary.txt** - A summary report generated by the CLI for a sample startup.
3. **founder_data.csv** - A CSV file containing startup data, including funding details.
4. **founder_focus_project.ipynb** - The Jupyter Notebook used for the project.

## How to Use

1. Clone the repository.
2. Run `founder_focus.py` to start analyzing startup investment data.
3. Use the provided functions to categorize startups, assess their readiness for acquisition, and generate reports.
